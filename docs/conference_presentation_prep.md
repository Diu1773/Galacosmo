# GalaCosmo 학회 발표 준비 노트 (2026-03-06)

## 핵심 메시지
1. GalaCosmo는 회전 곡선 + SN Ia 허블 다이어그램을 하나의 GUI 워크플로로 통합한다.
2. 최근 최적화로 인터랙티브 피팅 반복 속도를 크게 높였다.
3. 과학적 해석(모형 비교, 잔차, chi2)을 실시간 의사결정 가능한 형태로 제공한다.

## 권장 슬라이드 구성 (12장)
1. 제목: 프로젝트명, 소속, 발표자
2. 연구 동기: DM halo 모델 비교와 우주론 파라미터 탐색 필요성
3. 시스템 개요: Rotation + Hubble 두 분석 모듈 구조
4. 데이터: SPARC/Table1·Table2, Union2.1 입력 파이프라인
5. 물리 모델: ISO/NFW, baryon 합성, 거리모듈러스 계산
6. UI/분석 흐름: 로드 -> 선택 -> 피팅 -> 잔차 해석
7. 최적화 전 문제: 파라미터 변경 시 반복 I/O/중복 계산
8. 최적화 후 설계: 갤럭시별 Table2 메모리 캐시 + Table1 lookup 사전화
9. 성능 결과: 반복 피팅 응답 시간 비교 그래프
10. 사례 결과: 대표 은하 1~2개 피팅 및 잔차 해석
11. 라이브 데모: 실제 조작 2분 시나리오
12. 결론/향후 계획: 자동 보고서, 배치 벤치마크, 테스트 확장

## 데모 시나리오 (2분)
1. `Rotation Curve` 탭에서 SPARC 파일 로드
2. 은하 선택 후 ISO/NFW 전환, M/L 슬라이더 조정
3. 잔차 패널 On/Off 비교로 모델 차이 강조
4. `Galaxy 3D View` 탭 전환으로 구조 시각화 연결
5. `Hubble Diagram` 탭에서 참조 우주론/모델곡선 비교

## 성능 측정 기록 (로컬, 2026-03-06)
- 데이터: `data/SPARC_Table2.txt`
- 비교 항목:
  - 기존 패턴: `read_table2(path, galaxy)`를 피팅마다 재호출
  - 개선 패턴: 로드 1회 후 `ID_key` 기반 메모리 필터/캐시 재사용
- 측정 결과 (8개 은하, 각 15회 평균):
  - 파일 재파싱: `0.0693 s / fit`
  - 메모리 필터: `0.000835 s / fit`
  - 체감 예상 개선: 약 `83x`
